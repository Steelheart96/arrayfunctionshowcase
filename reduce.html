Optional: currentIndex, array, and initializeValue

What is ".reduce()" and what does it do?

A function for arrays.

This function returns a single value.

It iterates through each item in an array.

You can pass it a function that is executed each iteration.

Reduce function combines all values from an array in some way.

Learn More at developer.mozilla.org.
